# Trail Gear Journal

Premium static magazine site for US outdoor, camping, and overlanding gear.

## Stack

- Zero-dependency Node static generator
- Plain HTML, CSS, and JavaScript
- Cloudflare Pages-ready output in `dist/`

## Commands

```bash
npm run build
npm run verify
npm run dev
```

Cloudflare Pages build command: `npm run build`

Cloudflare Pages output directory: `dist`
